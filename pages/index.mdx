# Introduction

Welcome to Opexa Portal SDK documentation!

## What is Opexa Portal SDK?

The **Opexa Portal SDK** is a powerful and flexible library designed to simplify integration with the **Opexa**. It provides developers with a comprehensive set of tools and functions to seamlessly integrate Opexa's gaming platform features into their applications.
